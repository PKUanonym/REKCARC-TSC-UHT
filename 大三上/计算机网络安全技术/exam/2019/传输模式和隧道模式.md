传输模式和隧道模式

|      | 传输模式                                 | 隧道模式                             |
| ---- | ---------------------------------------- | ------------------------------------ |
| 目的 | 为上层协议提供保护<br />对IP包载荷的保护 | 对整个IP包提供保护                   |
| ESP  | IP载荷，不包括报头                       | 整个内部IP包，包括报头               |
| AH   | IP载荷和报头的选中部分                   | 整个内部IP包<br />外部报头的选中部分 |
