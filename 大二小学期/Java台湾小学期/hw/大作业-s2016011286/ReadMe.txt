运行方法：（建议通过命令行打开）
	cd到当前目录
	java -jar GreedSnake.jar 即可运行
运行环境：mac pro OSX 
测试环境：本地测试
操作方法：
	1、输入用户名密码注册-->登陆-->进入游戏初始界面（输入不支持中文）
	2、选择StartGame开始游戏
	3、server建立服务器，client连接输入IP与端口的服务器，同步完成后游戏将开始
	4、操作为WASD/上下左右均可，己方的蛇头为红色，对方为灰色，便于区分，其他基础功能参见大作业报告
备注：我没有把class文件单独放到一起，助教看代码的话在文件夹中搜索.java即可，麻烦您了
