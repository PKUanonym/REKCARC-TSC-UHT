These features are the content-based features used in:

Carlos Castillo, Debora Donato, Aristides Gionis,
Vanessa Murdock, Fabrizio Silvestri:
"Know your Neighbors: Web Spam Detection using the Web Topology".
To appear in ACM SIGIR 2007.
(Was DELIS technical report DELIS-TR-0458, 2006.)
http://www.dcc.uchile.cl/~ccastill/papers/cdgms_2006_know_your_neighbors.pdf

It was possible to compute content-based features only for 8,944 hosts that
appeared the .graph files and had their home page and page with the maximum
pagerank with content in the .warc file.
